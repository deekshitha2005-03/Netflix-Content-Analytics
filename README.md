# 🎬 Netflix Content Analytics Dashboard

## 📌 Project Overview

This project is an interactive **Netflix Content Analytics Dashboard** developed using **Power BI**. It provides comprehensive insights into Netflix's Movies and TV Shows by analyzing ratings, genres, production countries, runtime, cast members, directors, and release trends.

The dashboard enables users to explore Netflix content through interactive filters and visualizations, making it easy to identify trends and gain valuable business insights.

## 📈 Key Insights

- Drama is the most common genre on Netflix.
- Movies account for the majority of Netflix's content.
- The United States contributes the highest number of titles.
- IMDb ratings vary across genres and content types.
- Content production has grown significantly after 2015.
- TV-MA is the most common age certification.
- Certain cast members and directors appear frequently across multiple titles.
